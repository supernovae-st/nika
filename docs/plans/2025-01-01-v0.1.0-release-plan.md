# Nika v0.1.0 Release Plan

> Date: 2025-01-01
> Status: Planning

## Vision

Nika est un DAG workflow runner pour tâches AI. v0.1.0 est le premier release public après polish.

## Strategy

```
Repos GitHub:
├── nika-og        → Archive (legacy CLI)
├── nika-spec-og   → Archive (legacy spec)
└── nika           → NEW (v0.1.0+, clean history)

Visibility: Private → v0.1.0 stable → Public
Versioning: SemVer (0.1.0 → 0.2.0 → 1.0.0)
```
